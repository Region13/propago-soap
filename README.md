# Propago Soap API Wrapper

## Install

Via Composer

``` bash
$ composer require esc13/propago-soap
```

## Requirements

The following versions of PHP are supported by this version.

* PHP 5.4 - PHP 7.1
* phpsoap module
* PHP 5.5
* PHP 5.6
* PHP 7.0
* PHP 7.1

## Documentation

API has soap [documentation](http://test-api.mypropago.com/public/orderservice.asmx).
